# homebrew-tap

Homebrew tap for Towards the Cloud CLI tools.

## Install

```bash
brew tap towardsthecloud/tap
```

### Cloudburn

CLI for cloud cost optimization.

```bash
brew install towardsthecloud/tap/cloudburn
```

### awstbx

Unified CLI for AWS infrastructure automation.

```bash
brew install --cask towardsthecloud/tap/awstbx
```
